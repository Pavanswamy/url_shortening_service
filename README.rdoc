== URL Shortning Service

A Sample Application for Url Shortening Service. which can the track the traffic of the url.

* Ruby version > 2 +

* Rails version > 4 +

* Database creation
  > rake db:create
  > rake db:migrate  

* Services (job queues)
  Integrate any library for backgroud Jobs for disable dummy links(i..e., No Trafic generated) links

* You can View the app here https://url-shortning-service.herokuapp.com/
